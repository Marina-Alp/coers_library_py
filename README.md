Пакет предоставляет инструменты для работы с книгами, библиотеками и читательскими списками:

- cоздание книг (Название, атор, дата);
- создание библиотек (Название):
- создание читательских списков (Имя читатиля);
- изменять книгу и удалять;
- сравнение книги;
- добавлять и удалять книги в библиотеки;
- искать по названию и автору книги в библиотеке;
- сортировать книги в библиотеки по названию, автору и дате;
- добовлять и удалять книги из читательского списка;
- проверяет доступность книг в библиотеке;
- форматированный вывод данных.
